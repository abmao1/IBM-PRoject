5 4. Reproducible Chart from the Presentation

Below is an example of code that creates a reproducible chart.

#| label: reproducible-chart
#| echo: true
#| message: false
#| warning: false

# Load required library
library(ggplot2)

# Create simulated data
set.seed(8675309)
data <- data.frame(
  perceived_value = rnorm(100, mean = 6.5, sd = 1.2),
  willingness     = rnorm(100, mean = 6.0, sd = 1.5)
)

# Generate a scatter plot with linear fit
ggplot(data, aes(x = perceived_value, y = willingness)) +
  geom_point(alpha = 0.6) +
  geom_smooth(method = "lm", color = "steelblue") +
  labs(
    title = "Perceived Value vs. Willingness to Subscribe",
    x = "Perceived Bundle Value",
    y = "Willingness to Subscribe"
  ) +
  theme_minimal()

6 5. Three Advanced Wrangling Functions

Below are three functions from different categories (joining, pivoting, and variable creation). Each code example includes a short data frame and outputs the result.

{r}#| label: advanced-wrangling-examples
#| echo: true
#| message: false

library(dplyr)
library(tidyr)

## A. left_join() ‚Äì Use when merging datasets by key, keeping all from the left.

df_users <- data.frame(
  user_id = c(1, 2, 3),
  name    = c("Alice", "Bob", "Charlie")
)

df_orders <- data.frame(
  user_id     = c(1, 2),
  order_total = c(25, 40)
)

df_combined <- left_join(df_users, df_orders, by = "user_id")

cat("Output from left_join():\n")
print(df_combined)


## B. pivot_longer() ‚Äì Use when converting wide data to long format.

df_scores <- data.frame(
  student = c("A", "B"),
  math    = c(90, 85),
  english = c(88, 92)
)

df_long <- pivot_longer(
  df_scores,
  cols = math:english,
  names_to = "subject",
  values_to = "score"
)

cat("\nOutput from pivot_longer():\n")
print(df_long)


## C. mutate() ‚Äì Use when creating new variables/columns.

df_sales <- data.frame(
  price    = c(10, 20, 30),
  quantity = c(1, 3, 2)
)

df_sales <- df_sales %>% mutate(total = price * quantity)

cat("\nOutput from mutate():\n")
print(df_sales)
